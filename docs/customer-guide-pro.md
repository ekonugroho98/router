# Cortex AI — Quick Start Guide
### Pro Plan (30 Hari)

---

Selamat datang di Cortex AI Pro! Kamu punya akses penuh ke AI assistant dengan semua model, web automation, dan dedicated server.

---

## Step 1: Dapatkan Kode Aktivasi

Setelah pembayaran dikonfirmasi, kode dikirim via WhatsApp/Telegram.

Format kode: `CORTEX-XXXX-XXXX`

Belum terima kode? Chat admin: **@ekonugroho98** (Telegram)

---

## Step 2: Aktivasi Akun

1. Buka: **https://9router.cortex-ai.my.id/customer/activate**
2. Masukkan kode aktivasi
3. Isi email + password
4. Klik **"Buat Akun & Aktifkan"**

---

## Step 3: Setup Bot Telegram

**A. Buat Bot:** Buka @BotFather → kirim `/newbot` → copy Bot Token

**B. Chat ID:** Buka @userinfobot → kirim pesan → copy Id

**C. Paste** di halaman aktivasi → Klik Aktifkan

Panduan lengkap + screenshot: **https://docs.cortex-ai.my.id/setup/telegram-bot**

---

## Step 4: Mulai Chat!

Buka bot di Telegram → kirim `/start` → coba chat!
